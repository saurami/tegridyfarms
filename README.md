# Tegridy Farms

Single Page Application based on South Park's Tegridy Farms

## Getting Started

+ Test HTTP server

  `make test_all`

+ Run HTTP server

  `go run server.go`

+ To view content, navigate to

  1. Homepage (Image): http://localhost:8080/home
  2. Outdoor  (HTML):  http://localhost:8080/outdoor
